# 🛒 Amazon Grocery Demand Forecasting  
### 📊 Time Series Forecasting using ARIMA, Prophet & XGBoost

This project focuses on building a **demand forecasting system** for grocery products using advanced **time series and machine learning models**.  
The goal is to help optimize inventory management and reduce issues like stockouts and overstocking.

---

## 🎯 Real-World Scenario

Amazon Fresh often faces challenges in predicting customer demand accurately.  
Uncertain demand can lead to:

- ❌ Stockouts (lost sales)  
- ❌ Excess inventory (increased costs)  

This project aims to solve these problems using data-driven forecasting techniques.

---

## 🎯 Project Goal

To develop a **reliable demand forecasting system** using multiple models and compare their performance to:

- Improve inventory planning  
- Reduce operational inefficiencies  
- Support better business decision-making

## 📦 Dataset Overview

- **Source:** Kaggle - Store Sales Time Series Forecasting  
- **Description:** Daily sales data from multiple grocery stores in Ecuador  
- **Features Included:**
  - Sales data  
  - Holidays  
  - Oil prices  
  - Promotions  

### 🔹 Subset Used
- Store ID: 1  
- Product Category: GROCERY I

## 📌 Project Objectives

- Perform data cleaning and preprocessing  
- Explore historical sales trends  
- Analyze:
  - Trend  
  - Seasonality  
  - Holiday effects  
- Build and compare forecasting models:
  - ARIMA  
  - Prophet  
  - XGBoost  
- Evaluate models using RMSE, MAE, and MAPE  
- Discuss scalability to all stores and product categories
